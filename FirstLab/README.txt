ЗАДАЧА:
	- Cоздать проект в Android Studio, создать для него 3 build type (debug, prodDebug и release);
	- Cоздать product flavors 3 штуки таким образом, чтобы в зависимости от выбора флейвора менялся UI-элемент на экране (бэкграунд, цвет текста или что-то другое на ваш выбор).

РЕШЕНИЕ:
	- Создано 3 Build Types (debug, prodDebug, release);
	- Создано 3 Flavor Types (demo, free, pro);

	demo:
		- Иконка приложения с лейблом "DEMO"
		- "strings.xml" содержит уникальную текстовую информацию
		- "colors.xml" содержит красный цвет

	free:
		- Иконка приложения с лейблом "FREE"
		- "strings.xml" содержит уникальную текстовую информацию
		- "colors.xml" содержит голубой цвет

	pro:
		- Иконка приложения с лейблом "PRO"
		- "strings.xml" содержит уникальную текстовую информацию
		- "colors.xml" содержит зеленый цвет
